HTML and Layouts

HTML's role with regard to page layout is not as significant as it was before CSS was introduced, but there are still some important concepts to understand. The first is that every HTML element that appears on a web page is contained within a rectangle or box.

Understanding that all elements on a page occupy a certain rectangular size is fundamental to understanding layouts. One of the other important concepts is that certain types of elements will begin on new lines by default (occupying the width of their container) while other types of elements don't start on a new line and only take up the width of their contents. These are known as block-level and inline elements respectively and will be the next topics discussed.
