Basic Positioning

CSS allows elements to be positioned relative to how they would appear in normal flow, removed from normal flow completely and positioned with respect to the containing element, or positioned with respect to the browser window. The most difficult part of learning about positioning is understanding the unintuitive naming of the types of positioning. Let's take a look at the different values for the position property and what they mean.

In the Flow

static
Static is the default value for the position property. This means the element will be positioned as it occurs in normal document flow.
relative
Relative positioning moves the element relative to its default (static) position. Relative positioning keeps the element in normal flow and the element still occupies its default position, it's just repositioned relative to the default.
